# Installation

## Build

#### Prerequisites

* [Rust](https://www.rust-lang.org/tools/install)
* [Golang](https://golang.org/doc/install) &gt;= 1.13
* [protoc](http://google.github.io/proto-lens/installing-protoc.html) &gt;= 3.6.1
* [rustup](https://rustup.rs/)
* make and cmake

#### Build

```bash
make
```

#### Test

```bash
make test
```
